Αρχική Σελίδα

Εισαγωγή

Λίγα λόγια για το project.

Πρόκειται για ένα project παιχνιδιού περιπέτειας που δημιουργήθηκε στα πλαίσια τελικής εργασίας στο μάθημα Αντικειμενοστρεφής Προγραμματισμός. Το παιχνίδι τρέχει μόνο σε λειτουργικό σύστημα Linux η οποία είναι και η μοναδική απαίτηση.

Πριν την εγκατάσταση

Τι χρειάζεστε

  • NCurses Library
  • Make(build-essential)
  • C++11

Οδηγίες εγκατάστασης

(A) Repository

Μετακινηθείτε στον κατάλογο οπου επιθυμείτε να κάνετε clone και πληκτρολογηστε στο τερματικο

git clone https://github.com/GeorgeGRz/ncurses-game-project.git

(B) Συμπιεσμένο αρχείο

Αποσυμπιέστε το αρχείο στον επιθυμητό κατάλογο με το command

tar -xzvf ncurses-game-project.tar.gz

Μετακίνηση στον κατάλογο που εγινε clone

Στη συνέχεια πληκτρολογηστε

cd ncurses-game-project/ && make all

Τρέξτε το παιχνίδι.

Για να τρέξετε το παιχνίδι απλά πληκτρολογηστε ενώ βρίσκεστε στον ίδιο κατάλογο

./adventureGame
